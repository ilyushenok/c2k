# Sandboxed Google Play on GrapheneOS

For Play Store developer account verification. Fully reversible.

## Setup

1. Settings > System > Multiple users > Add user
2. Switch to the new profile: swipe down notification shade, tap avatar icon top-right
3. Open the **Apps** app (GrapheneOS app store, pre-installed)
4. Install **Google Play Store** — pulls in the full sandboxed suite automatically
5. Sign in to Play Store with your Google account
6. Install the **Play Console** app and complete verification

## Removal

Settings > System > Multiple users > select the profile > Delete

Everything in that profile is gone. Main profile is untouched.
